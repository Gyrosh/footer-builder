# Footer builder

## Install & Dev

Run `npm install` and then `npm run start`

## Configure

Fill out environment variables in `environment.ts` file:

```
export const environment = {
   production: false,
   flotiqApiUrl: 'https://api.flotiq.com',
   flotiqApiKey: 'FLOTIQ_API_KEY',
   mjmlApi: 'https://api.mjml.io',
   mjmlApplicationKey: 'MJML_API_KEY',
   mjmlPublicKey: 'MJML_PUBLIC_KEY'
};
```
